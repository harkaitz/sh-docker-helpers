# DOCKER HELPERS

## Help

docker-h-delete

    Usage: docker-h-delete CONTAINERS... : Kill containers and delete.

docker-h-exec

    Usage: docker-h-exec [CONTAINER [COMMAND]]

docker-h-stop

    Usage: docker-h-stop
    
    Stop all containers.

dsh

    Usage: dsh [CONTAINER [COMMAND]]
    
    An alias for docker-h-exec(1).
